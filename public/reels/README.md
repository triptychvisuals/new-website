# Project reels

These are the looping clips shown in the homepage grid cards.

`reel-01.svg … reel-08.svg` are animated placeholders. To use your own footage:

## Drag-and-drop (no local setup needed)

1. Go to **github.com/triptychvisuals/website**
2. Switch to branch **`claude/bold-franklin-lp9d4g`**
3. Open the **`public/reels`** folder
4. Click **Add file → Upload files**
5. **Drag your `.gif` / `.mp4` files here** and commit

Then tell Claude the filenames (e.g. `gherbo.gif`, `nardo.mp4`) and they'll be
wired into the grid and deployed.

Prefer short, muted `.mp4`/`.webm` loops over large `.gif`s for quality + speed.
